CSS Parts

Use ::part selectors to style internal elements safely.

Toggle Delete on a row to see ::part(deleted) styling.

ck-primitive-array.part-demo::part(list) { border: 1px dashed #94a3b8; }
ck-primitive-array.part-demo::part(row) { background: #fff; }
ck-primitive-array.part-demo::part(deleted) { opacity: 0.55; }
ck-primitive-array.part-demo::part(input) { border-color: #38bdf8; }
ck-primitive-array.part-demo::part(delete-button) { background: #fee2e2; }
ck-primitive-array.part-demo::part(remove-button) { background: #111827; }

::part(list)

Targets the list wrapper element.

::part(row)

Targets each row container.

::part(deleted)

Targets rows that are soft-deleted.

::part(input)

Targets the text input inside each row.

::part(delete-button)

Targets the delete/undo button.

::part(remove-button)

Targets the hard remove button.

Basic Usage

Minimal setup, core attributes, and essential behaviors.

Default Component

<ck-primitive-array></ck-primitive-array>

name Attribute

<ck-primitive-array name="Team"></ck-primitive-array>

The name is used in the greeting and as the form field name when submitting.

color Attribute

<ck-primitive-array name="Ocean" color="#0369a1"></ck-primitive-array>

items Attribute

<ck-primitive-array name="Priorities" items='["Launch", "Ship"]'></ck-primitive-array>

readonly Attribute

<ck-primitive-array readonly items='["Alpha"]'></ck-primitive-array>

Readonly disables editing and action buttons, but values remain visible.

disabled Attribute

<ck-primitive-array disabled items='["Locked"]'></ck-primitive-array>
